# **App Name**: LoveBloom Invites

## Core Features:

- Main Invitation Display: Clearly presents all essential wedding details including couple's names, event date, time, and venue.
- Personal Photo Gallery: Showcases a curated selection of romantic photographs of the couple.
- Digital RSVP Management: Allows guests to easily confirm their attendance, specify the number of guests, and note any dietary restrictions.
- Guest Wishes Wall: Provides a dedicated section for invitees to post congratulatory messages and well wishes for the couple.
- AI-Powered Greeting Assistant: Offers a tool to suggest thoughtful and personalized messages for guests to leave on the wishes wall, based on provided keywords or sentiments.
- Event Details Map Integration: Embeds an interactive map providing directions to all wedding event locations.

## Style Guidelines:

- Background color: A delicate, warm off-white (hsl(30, 15%, 98%), hex: #FAF8F6) to provide a soft, elegant canvas.
- Primary color: A sophisticated, muted gold-beige (hsl(30, 30%, 75%), hex: #D0C0AB) symbolizing warmth and enduring commitment, used for key headings and prominent details.
- Accent color: A soft, dusty rose (hsl(0, 40%, 65%), hex: #B97070) analogous to the primary, offering a touch of romantic contrast for call-to-action elements or highlighted text.
- Headline font: 'Playfair' (serif) for an elegant, high-end feel for couple's names and key dates. Body font: 'PT Sans' (sans-serif) for clear readability of event details and messages.
- Use minimalist and elegant line-art icons that evoke wedding themes, such as intertwined rings, subtle floral accents, and stylized heart shapes.
- Emphasize a clean, responsive layout with ample negative space to ensure content is easily digestible and visually appealing across all devices, maintaining an air of sophistication.
- Implement subtle, slow fade-in and slide-up animations for content sections as the user scrolls, creating a smooth and understated presentation flow.
